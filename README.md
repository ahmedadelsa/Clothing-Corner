# 🛒 Clothing Corner - Full E-Commerce System

## 🚀 Overview
Clothing Corner is a full-stack e-commerce platform built from scratch using ASP.NET Core Web API and a modern JavaScript frontend.

It includes authentication, product management, shopping cart, checkout, payment flow, and an admin dashboard.

---

## ✨ Features

### 👤 User
- Register & Login (JWT Authentication)
- Browse products
- Search & filter products
- Add to cart
- Update quantity / remove items
- Checkout & Payment (Demo)
- View orders

### 👑 Admin
- Add / Delete products
- Upload product images
- Add categories
- Manage orders & update status

### 🎨 UI
- Responsive design
- Dark mode 🌙
- Modern UI (Bootstrap + Vite)

---

## 🛠 Tech Stack

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication

### Frontend
- HTML, CSS, JavaScript
- Vite
- Bootstrap
